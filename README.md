# Jogo da Velha com IA Simples 🎮🤖
Projeto desenvolvido em .NET MAUI com o objetivo de criar um clássico Jogo da Velha contra uma Inteligência Artificial simples.

O projeto foi criado para fins de estudo e prática de:
Desenvolvimento Mobile/Desktop com .NET MAUI
Lógica de programação
Inteligência Artificial básica
Manipulação de interface gráfica com XAML
Estruturação de aplicações multiplataforma

# 🚀 Tecnologias utilizadas
C#
.NET MAUI
XAML
Visual Studio 2022

# 🧠 Funcionalidades
✅ Tabuleiro interativo
✅ Jogador contra IA
✅ Verificação de vitória
✅ Verificação de empate
✅ Reinício de partida
✅ Interface simples e intuitiva
